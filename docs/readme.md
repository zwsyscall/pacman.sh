# readme
### This website serves as a collection and blog for random thoughts I have
I primarily use this domain for email, so I'll try to be *somewhat* professional.

A lot of it is nonsensical and just me screaming into the void. Writing things down helps with processing them. Aside from letting me write and vent out my feelings without bugging people, I can try to keep myself accountable by posting about aspirations and goals here.

Over time, I might edit or remove posts if I think some of them are innapropriate or don't reflect my views anymore. For transparency's sake and me keeping me accountable, they can be found on this site's [github repo](https://github.com/zwsyscall/pacman.sh)
